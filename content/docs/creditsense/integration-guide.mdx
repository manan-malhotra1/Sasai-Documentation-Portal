## Integration flow

1. Provision sandbox access for lending and scoring capabilities.
2. Authenticate backend calls using Sasai-managed credentials.
3. Run eligibility and scoring checks before disbursement requests.
4. Subscribe to lifecycle events for approvals, disbursements, and repayments.
5. Promote tested loan flows to production.
