## Integration flow

1. Configure sandbox credentials and insurance product access.
2. Authenticate backend requests with Sasai-issued secrets or API keys.
3. Submit policy issuance and premium collection requests from trusted services.
4. Listen for claim and policy lifecycle webhooks.
5. Promote validated insurance journeys to production.
